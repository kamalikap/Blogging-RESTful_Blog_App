RESTful Routes

A table for all 7 RESTful Routes.

Name Path HTTP Purpose Mongoose Methods
Index /blogs GET List all blog post Blog.find()
New /blogs/new GET new blog post N/A
Create /blogs POST creat a new post, then redirect to somewhere Blog.create()
Show /dogs/:id GET Show info about a specific post Blog.findById()
Edit /blogs/:id/edit GET Edit a specific post Blog.findById()
Update /blogs/:id PUT Update the blog post and then redirect Blog.findByIdAndUpdate()
Destroy /dogs/:id DELETE Delete a post Blog.findByIdAndRemove()
